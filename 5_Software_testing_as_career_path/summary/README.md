- software testing adalah tes yang dilakukan untuk menemukan bug yang ada pada program.
  sebagai tester juga perlu memikirkan segala kemungkinan yang dapat terjadi ketika user menggunakan
  software yang kita tes

- attitude menjadi faktor yang paling penting dalam menentukan keberhasilan
  seseorang untuk menjadi seorang software tester

- diperlukan juga keahlian technical untuk menjadi seorang tester seperti
  dapat menggunakan automation tools
